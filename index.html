💡 The Big Idea

Variables — Your Code's Memory

Imagine you're making a game. Your player has a name, a score, and a number of lives. How does your programme remember all of that? With variables.

A variable is a named space in your computer's memory where you can store a piece of information. You give it a name, and you can use or change that information any time you need it.

📦
Think of it like this

Imagine a row of boxes with labels on them. One box is labelled name, another is labelled score, another is labelled lives. Each box holds something. You can open any box to look inside, change what's in it, or use it somewhere else. That's exactly what a variable does.

In Python, creating a variable is simple. You just write the name, an equals sign, and then the value you want to store:

Python · variables.py 
# Creating variables — three different data types
player_name = "Zara"         # a string (text)
player_score = 0              # an integer (whole number)
player_lives = 3              # another integer

# Using variables
print("Player:", player_name)  # prints → Player: Zara
print("Score:", player_score)  # prints → Score: 0

# Changing a variable
player_score = player_score + 10
print("New score:", player_score)  # prints → New score: 10

Notice that player_score = player_score + 10 takes the current value, adds 10, and stores the result back in the same box. The box now contains 10 instead of 0.

✨ Did You Know?

Variables Are Everywhere

🧠

Fun Fact

The first programming language to use variables as we know them today was FORTRAN, created in 1957. That's over 65 years ago! Back then, programmers had to manually punch holes in cards to write code — but the concept of "store a value, give it a name, use it later" has never changed.

🌍 Real-World Connection

Right now, as you scroll through TikTok or Instagram, there are variables running behind the scenes — current_user, video_likes, scroll_position. Every app you use on your phone is full of thousands of variables working together, all doing exactly what you just learned.
